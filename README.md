## Getting Started

1. Clone the repository

```
git clone https://github.com/jitthing/FundTasy.git
```

2. Install the necessary dependencies

```
npm install
```

3. Run backend in development environment

```
cd backend
```
```
npm run devStart
```

4. Start frontend in development environment

```
cd frontend
```
```
npm start
```
